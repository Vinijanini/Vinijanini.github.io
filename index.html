Sobre Mim

Sou apaixonado por tecnologia e desde 2020 venho me aprofundando no mundo da programação. Com um forte raciocínio lógico e habilidade para resolver problemas, encontrei na tecnologia uma forma de transformar ideias em soluções práticas e eficientes. Atualmente, estou cursando Análise e Desenvolvimento de Sistemas, o que tem me proporcionado uma base sólida e constante aprendizado.

Tenho experiência no desenvolvimento com Python, incluindo a criação de APIs com Flask, manipulação de banco de dados utilizando SQLite3, SQLAlchemy e SQL, além de trabalhar com bibliotecas como Pandas para análise de dados. Minha compreensão de programação orientada a objetos me permite construir soluções escaláveis e bem estruturadas.

Além disso, também possuo conhecimentos em HTML, CSS e JavaScript, o que me capacita a desenvolver sites e páginas web responsivos. Tenho experiência em consumo e manipulação APIs, utilizando JavaScript para lidar com dados JSON de forma eficaz.

Consigo criar automações utilizando bibliotecas como PyAutoGUI e Selenium, o que me permite automatizar tarefas repetitivas em sistemas, interagir com interfaces gráficas e realizar testes automatizados em sites, melhorando a produtividade e a qualidade dos processos.

Estou sempre em busca de novos desafios e oportunidades para crescer como programador, aprimorando minhas habilidades e contribuindo para projetos inovadores.
